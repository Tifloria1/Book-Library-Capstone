## Introduction
This is my frontend capstone project for the ALX program.
The app allows users to search for books and view details using the Open Library API.

## Features
- Search books by keyword
- Display book covers, titles, and authors
- View book details
- Loading, error, and no-results handling

## Technologies
- React (Vite)
- JavaScript
- Open Library API

## Run locally
npm install
npm run dev

## API
https://openlibrary.org/search.json

## Known limitations
Some books may not have cover images.
